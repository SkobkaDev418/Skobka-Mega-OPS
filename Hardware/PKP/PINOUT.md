[EN]
Complete connection table for Arduino Mega. Pins 7 and 8 have been removed as requested.

### 1. RFID Module (MFRC522)
* **53 (SDA), 52 (SCK), 51 (MOSI), 50 (MISO), 49 (RST)**.
* **Power:** 3.3V and GND.

### 2. Audio & Notification
* **18 (TX1)**: DFPlayer RX (connection via 2kOhm resistor).
* **19 (RX1)**: DFPlayer TX.
* **6**: System Buzzer (Siren).

### 3. Monitoring Zones (Analog)
* **A0 - A5**: Fire Zones 1-6.
* **A6 - A7**: Security Zones 7-8.
* *Note: All zones support Short Circuit and Open Circuit detection.*

### 4. Status & Control (Digital)
* **22 - 29**: Visual indicators for Zones 1-8.
* **5**: "EXIT" Sign Relay control.
* **4**: System Armed Status LED.
* **13**: Onboard System LED (Status blink).
