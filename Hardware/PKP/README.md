This section contains all technical documentation for the **Control Panel (CP)**. It is the central hub of the system that collects sensor data, processes alarms, and manages notifications.

### What the Control Panel does:
* **Monitoring:** Constantly polls 8 zones for short circuits or wire breaks.
* **Security:** Allows arming and disarming the system via RFID.
* **Notification:** Activates the siren and voice guidance during an alarm.
* **Indication:** Displays the status of each zone and the system state on the panel.

### Folder Contents:
* **Gerber:** Files for PCB manufacturing.
* **STL:** 3D models for the enclosure and internal mounts.
* **Schematics:** Internal wiring and component connection diagrams.
* **BOM:** Bill of Materials (list of required modules and components).
